# Tour Images

This directory contains tour images for Atash Travel.

## Expected Structure

```
tours/
├── van-tour.jpg          # Main card image for Van tour
├── antalya-tour.jpg      # Main card image for Antalya tour
├── istanbul-tour.jpg     # Main card image for Istanbul tour
├── trabzon-tour.jpg      # Main card image for Trabzon tour
├── van-hero.jpg          # Hero section image for Van tour detail page
├── antalya-hero.jpg      # Hero section image for Antalya tour detail page
├── istanbul-hero.jpg     # Hero section image for Istanbul tour detail page
├── trabzon-hero.jpg      # Hero section image for Trabzon tour detail page
├── van-og.jpg            # OpenGraph image for Van tour (1200x630)
├── antalya-og.jpg        # OpenGraph image for Antalya tour (1200x630)
├── istanbul-og.jpg       # OpenGraph image for Istanbul tour (1200x630)
├── trabzon-og.jpg        # OpenGraph image for Trabzon tour (1200x630)

├── antalya-1.jpg through antalya-6.jpg   # Gallery images for Antalya tour

```

## Image Requirements

### Card Images (listing page)
- **Dimensions**: 800x600px (4:3 aspect ratio)
- **Format**: JPG optimized
- **File size**: < 200KB

### Hero Images (detail page)
- **Dimensions**: 1920x1080px (16:9 aspect ratio)
- **Format**: JPG optimized
- **File size**: < 300KB

### OpenGraph Images (social sharing)
- **Dimensions**: 1200x630px
- **Format**: JPG optimized
- **File size**: < 200KB

### Gallery Images
- **Dimensions**: 1200x900px (4:3 aspect ratio)
- **Format**: JPG optimized
- **File size**: < 250KB each

## Placeholder Behavior

Until actual images are uploaded, the tour cards and pages will display:
- Gradient placeholders with camera icon
- No broken image errors
- Proper spacing and layout maintained

## Adding New Images

1. Optimize images before upload
2. Use descriptive filenames matching tour slugs
3. Ensure proper aspect ratios
4. Compress for web (use tools like TinyPNG or ImageOptim)
